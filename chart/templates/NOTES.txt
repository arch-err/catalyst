  http://{{ .Values.ingress.host }}/

{{- else }}

To access the application, forward the service port:

  kubectl port-forward svc/{{ include "catalyst.fullname" . }} 8080:{{ .Values.service.port }} -n {{ .Release.Namespace }}

Then visit http://localhost:8080
{{- end }}

{{- if .Values.secrets.useExisting }}

IMPORTANT: This chart expects pre-existing secrets in the {{ .Release.Namespace }} namespace:

  1. "catalyst-secrets" with keys: CATALYST_SECRET, COOKIE_SECRET
  2. "{{ include "catalyst.sshKeySecretName" . }}" with the SSH private key

Create them before the pods will start successfully:

  kubectl create secret generic catalyst-secrets \
    --from-literal=CATALYST_SECRET=<your-secret> \
    --from-literal=COOKIE_SECRET=<your-cookie-secret> \
    -n {{ .Release.Namespace }}

  kubectl create secret generic {{ include "catalyst.sshKeySecretName" . }} \
    --from-file=id_ed25519=<path-to-your-ssh-key> \
    -n {{ .Release.Namespace }}
{{- end }}
